# Contributing to BioCypher Registry

Thank you for your interest in contributing!

## How to Contribute
- Open an issue to discuss your idea or bug before submitting a pull request.
- Fork the repository and create a new branch for your changes.
- Write clear, concise commit messages.
- Ensure your code passes all tests and follows project style guidelines.
- Submit a pull request with a clear description of your changes.

## Code of Conduct
By participating, you agree to follow our [Code of Conduct](CODE_OF_CONDUCT.md).

For more details, see the README or open an issue with your questions.
